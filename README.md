# ASCEND | Precision Aerial Intelligence

A clean, modern landing page for ASCEND—engineering-first drone solutions & surveying for Navi Mumbai.

**Built with:**
- HTML5 + Tailwind CSS (via CDN)
- Vanilla JavaScript (no dependencies)
- Apple-inspired design system

## Live Site

**GitHub Pages:** https://harsh-pandhe.github.io/Fly/

## Structure

```
Fly/
├── index.html      # Main landing page
├── styles.css      # Custom styles
├── scripts.js      # Interactive features
└── README.md       # This file
```

## Features

- **Responsive Hero:** Full-width hero with fade-in animations
- **Bento Grid Technology Showcase:** Built-in-house hardware specs
- **Interactive Hardware Anatomy:** Hover over drone image to explore components
- **Services Showcase:** Surveying, Inspection, Custom Development
- **Modal System:** Click POI markers for detailed hardware info
- **Contact Section:** Email, phone, GitHub, LinkedIn links

## Customization

### Colors & Fonts

Edit the Tailwind config in `index.html` to customize:
- Primary blue: `#0071e3`
- Background: `#f5f5f7`
- Typography: Inter (Google Fonts)

### Images

Update Unsplash image URLs in `index.html`:
- Drone surveying (hero)
- Compute module (Raspberry Pi)
- Inspection example
- Drone prototype

### Content

Edit text directly in `index.html` sections:
- Hero copy
- Service descriptions
- Contact details

## Deployment

### GitHub Pages (Current)

Already enabled. Site auto-publishes from `main` branch.

To update: commit and push to `main`:
```bash
git add .
git commit -m "Update content"
git push origin main
```

### Custom Domain

1. Go to repo settings → Pages
2. Under "Custom domain," add your domain
3. Update DNS records pointing to GitHub Pages IP
