# ASCEND | Precision Aerial Intelligence

ASCEND is a modern landing page showcasing engineering-first aerial intelligence: autonomous flight systems, drone operations, and precision data capture.

## Live Site

- Vercel: https://fly-delta-khaki.vercel.app/
- GitHub Pages: https://harsh-pandhe.github.io/Fly/

## Highlights

- Apple-inspired layout with refined typography and spacing
- Animated hive visualization and 3D flight scene
- Modular sections for technology, anatomy, services, and process
- Responsive layout with reduced motion support

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- Tailwind CSS via CDN
- Fonts: Sora + Manrope (Google Fonts)

Open `index.html` in a browser. No build step required.

## Deployment

### Vercel

- Framework: Other
- Build command: none
- Output directory: /

### GitHub Pages

Push to `main` and enable Pages → Deploy from branch.
