# Contributing

Thanks for your interest in contributing to ASCEND.

## How to Contribute

1. Fork the repository and create your branch from `main`.
2. Make your changes with clear, focused commits.
3. Ensure the site still loads locally.
4. Open a pull request with a short summary and screenshots if the UI changed.

## Development

- This is a static site. No build step is required.
- Open `index.html` in a browser to preview changes.

## Style Guide

- Keep HTML semantic and accessible.
- Avoid large third-party dependencies.
- Keep animations subtle and respectful of `prefers-reduced-motion`.

## Reporting Issues

Please use GitHub Issues and include:
- Steps to reproduce
- Screenshots if relevant
- Browser and OS details
